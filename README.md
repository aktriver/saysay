# Saysay

Saysay is a template project of Electron which uses vite as transpiler.

## Work in progress

I'm going to write the feature which makes siri speak any words for a sample implementation.

## Setup

```bash
yarn install --frozen-lockfile
```

## Dev

```bash
yarn dev
```

## Build (Production)

```bash
yarn build
# yarn start
```

## Packaging

```bash
yarn build
yarn dist
```
